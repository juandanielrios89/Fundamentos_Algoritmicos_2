# Página Web del Gimnasio FitLife

Esta es una página web simple y responsiva para un gimnasio ficticio llamado FitLife. El sitio incluye secciones para inicio, acerca de, servicios y contacto.

## Estructura del Proyecto

- `index.html`: Archivo principal HTML con la estructura de la página
- `style.css`: Hoja de estilos CSS para el diseño y la responsividad
- `script.js`: Archivo JavaScript para interactividad (formulario de contacto, navegación suave, animaciones)

## Características

- Diseño responsivo que se adapta a dispositivos móviles y de escritorio
- Navegación suave entre secciones
- Formulario de contacto funcional (con alerta de confirmación)
- Animaciones de entrada para las secciones
- Paleta de colores moderna y atractiva

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador web preferido.
2. Navega por las diferentes secciones usando el menú de navegación.
3. Completa el formulario de contacto para probar la funcionalidad.

## Personalización

Puedes personalizar el sitio cambiando:
- Los colores en `style.css`
- El contenido del texto en `index.html`
- La lógica del JavaScript en `script.js`
- Agregar imágenes reales reemplazando el placeholder en la sección de inicio

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
